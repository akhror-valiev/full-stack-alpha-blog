
<h1>Full-Stack Blog App</h1>
<h4>Set Up Project</h4>

- Clone the GitHub repo
- bundle install
- Configure your username and password in the database.yml
- rails db: create
- rails db: migrate
- rails s
- OR
- rails db: setup
- rails s
